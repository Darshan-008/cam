# 🖨️ Print Queue Management System

A full-stack DevSecOps-enabled application to manage student print queues in real-time, built using the MERN stack and powered by CI/CD tools like GitHub Actions, Jenkins, Docker, and Terraform.

---

## 🚀 Features

- Student-friendly print queue management system
- Real-time file upload and queue update
- Full-stack MERN architecture with MySQL
- Automated CI/CD pipeline with infrastructure provisioning
- Dark and light mode UI with Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend
- **React.js**
- **Tailwind CSS**
- **Vite** (Frontend build tool)

### Backend
- **Express.js**
- **MySQL** (Database)
- **Node.js**

### DevOps & CI/CD
- **GitHub Actions** – for CI & Terraform formatting and planning
- **Jenkins** – for automated builds and deployments
- **Docker** – containerization for frontend & backend
- **Terraform** – infrastructure as code (IaC)
